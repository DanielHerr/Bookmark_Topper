# Bookmark Topper
Moves newly created bookmarks to the top of their folder.

Versioning: Major.minor.package.store

Packages:
1 - Mv2 CRXv2 (Chromium 56-75)
2 - Mv2 CRXv3 (Chromium 64-138) or Mv2 XPI (Firefox 52+)
3 - Mv3 (Chromium 88+ or Firefox 109+)

Stores:
0 - none, self hosted
1 - Chrome Web Store
2 - Edge Add-ons
3 - Firefox Add-ons
4 - Opera addons
