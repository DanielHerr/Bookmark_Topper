# Bookmark Topper
Moves newly created bookmarks to the top of their folder.

Versioning: Major.minor.manifest/package.store

Packages:<br>
1 - Mv2 CRXv2 (Chromium 56-75)<br>
2 - Mv2 CRXv3 (Chromium 64-138) or Mv2 XPI (Firefox 53+)<br>
3 - Mv3 (Chromium 88+)<br>

Stores:<br>
0 - self hosted Chromium<br>
1 - self hosted Gecko<br>
2 - Chrome Web Store<br>
3 - Edge Add-ons<br>
4 - Opera addons<br>
5 - Firefox Add-ons<br>
